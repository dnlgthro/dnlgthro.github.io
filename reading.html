On web design principles

The following readings contain principles to follow when designing and writing for the web. It was these readings (as well as others) that guided my creation and design of this protfolio.

  1. User Experience Deliverables by Peter Morville

  2. Legibility, Readability, and Comprehension: Making Users Read Your Words by Jakob Nielsen

  3. Web Design is 95% Typography by Oliver Reichenstein

  4. How To Use HTML Meta Tags by Kristine Schachinger

  5. W3Schools for help with HTML and CSS
